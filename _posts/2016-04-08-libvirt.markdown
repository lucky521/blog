# Libvirt是什么

它是为了统一化操作而设计的“中间件”。
每种虚拟化hypervisor产品都有自己的管理命令，方法各不相同。
libvirt配置在hypervisor上层，可以让管理kvm、xen、esx、qemu等虚拟化hypervisor的操作一致化。


# libvirt的功能

- 远程操作hypervisor。
- VM管理：开关、挂起、迁移、热插拔。
- 存储管理：创建各种格式（qcow2，VMDK）的文件镜像，挂载NFS共享，列出现有的LVM卷组，创建新的LVM卷组和逻辑卷，原始磁盘分区等等。
- 网卡管理、NAT、路由管理

# libvirt支持的hypervisor drivers

- LXC - Linux Containers
- OpenVZ
- QEMU
- Test - Used for testing
- UML - User Mode Linux
- VirtualBox
- VMware ESX
- VMware Workstation/Player
- Xen
- Microsoft Hyper-V
- IBM PowerVM (phyp)
- Virtuozzo
- Bhyve - The BSD Hypervisor



# libvirt四件套

下面是libvirt相关的四件套。

- libvirt的核心是提供虚拟化管理的一堆库。调用其中的API就可以完成所提供的功能。
- virsh命令提供了交互式的操作方法。
- virt-manager是virsh的可视化管理程序。
- virt-install命令提供了命令行安装guest os的功能。

# 编译安装libvirt

使用libvirt来进行虚拟化平台管理，并不需要对hypervisor做什么修改。我们在远程的一台linux上安装libvirt就可以了，然后用这台机器去管理各个hypervisor。

```
> #sudo apt-get install gcc make pkg-config libxml2-dev libgnutls-dev libdevmapper-dev libcurl4-gnutls-dev python-dev libpciaccess-dev libxen-dev libnl-dev libyajl-dev uuid-dev
> tar -xzf libvirt-1.X.X.tar.gz
> /configure --help
> ./configure --prefix=/usr --localstatedir=/var --sysconfdir=/etc --with-esx=yes --with-xen=yes  --with-lxc
# 编译的时候要选择自己要使用的hypervisor driver
> sudo make
> sudo make install
> virsh --version
> libvirtd --version
> sudo initctl start libvirt-bin
```
