# 什么是机器学习系统？

我觉得机器学习系统作为一个软件系统，它不是一个独立的软件系统，而总是作为一个工具服务于另一个更大的软件系统。比如用于预测一件事情发生的概率，或是用于预测一个事物所属的归类，而被安放在某种识别系统、推荐系统、搜索排序系统、广告系统之中。
